# Revi Dashboard

Um dashboard moderno de CRM e Marketing para gerenciar campanhas, clientes e métricas.

## Tecnologias Utilizadas

- React
- TypeScript
- Tailwind CSS
- Recharts (Gráficos)
- Lucide React (Ícones)
- date-fns (Manipulação de datas)

## Como executar

1. Instale as dependências:
```bash
npm install
```

2. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```
